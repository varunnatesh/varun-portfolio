===============================================
           START HERE - STEP 1
===============================================

1. Open this folder in Command Prompt:
   C:\vaunportfolio

2. Run these commands ONE BY ONE:

   git init
   git add .
   git commit -m "Deploy my portfolio"

3. Go to GitHub:
   - Visit: https://github.com/new
   - Repository name: varun-portfolio
   - Make it PUBLIC
   - DON'T add README
   - Click "Create repository"

4. GitHub will show you commands. Use these instead:
   (Replace YOUR_GITHUB_USERNAME with your actual username)

   git remote add origin https://github.com/YOUR_GITHUB_USERNAME/varun-portfolio.git
   git branch -M main
   git push -u origin main

===============================================
           STEP 2 - DEPLOY
===============================================

1. Go to: https://railway.app
2. Click "Login with GitHub"
3. Click "New Project"
4. Click "Deploy from GitHub repo"
5. Select "varun-portfolio"
6. Railway will build and deploy automatically!

===============================================
           STEP 3 - CONFIGURE
===============================================

In Railway dashboard:

1. Click "Variables" tab
2. Click "New Variable" and add:
   
   DEBUG
   False

   SECRET_KEY
   varun-secret-key-2026-portfolio-production

   (Railway generates domain automatically)

3. Click "Settings" → "Networking"
4. Click "Generate Domain"
5. Copy your URL

6. Go back to "Variables"
7. Add one more:
   
   ALLOWED_HOSTS
   [paste your railway domain here]

8. In Railway, go to deployment
9. Open "Console" tab
10. Run:
    python manage.py createsuperuser
    
    Create admin account!
