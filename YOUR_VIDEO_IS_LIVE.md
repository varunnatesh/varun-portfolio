## 🎨 Customization Options

### **Want to adjust the effects?**

**Slow down animations:**
Edit `static/css/style.css`:
```css
animation: float-video 10s ease-in-out infinite; /* was 6s */
```

**Remove 3D tilt:**
```css
transform: perspective(1000px) rotateY(0deg); /* was -5deg */
```

**Change glow color:**
```css
background: radial-gradient(circle, rgba(YOUR_COLOR), transparent);
```

**Disable autoplay:**
Edit `templates/portfolio/home.html`:
Remove `autoplay` from video tag
